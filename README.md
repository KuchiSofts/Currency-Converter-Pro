# Currency Converter Pro

Professional currency converter Chrome extension with intelligent auto-conversion, beautiful themes, and real-time exchange rates.

## Features

- **Smart Currency Conversion** - Automatically detects and converts prices on web pages
- **40+ Currencies** - Support for all major world currencies
- **14 Beautiful Themes** - From dark mode to vibrant gradients
- **Right-Click Conversion** - Convert any price with a right-click
- **Auto-Convert Mode** - Automatically convert all prices on page load
- **Real-Time Rates** - Updated exchange rates from reliable sources

## Installation

1. Clone or download this repository:
   ```bash
   git clone https://github.com/KuchiSofts/Currency-Converter-Pro.git
   ```

2. Open Chrome and go to `chrome://extensions/`

3. Enable "Developer mode" (toggle in top right)

4. Click "Load unpacked" and select the extension folder

5. Done! Click the extension icon to start converting currencies

## Usage

### Quick Convert
- Click the extension icon
- Enter amount and select currencies
- See instant conversion

### Right-Click Conversion
- Browse any website
- Right-click on any price
- Select "Convert to [Your Currency]"
- Price is instantly converted

### Auto-Convert
- Open extension settings
- Enable "Auto-convert on page load"
- Select your target currency
- All prices convert automatically
